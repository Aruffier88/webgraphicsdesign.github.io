Tables and Forms with Bootstrap

Bootstrap offers lots of beautifully designed forms and tables that are easy to add to any exisiting website. The tables are easy to edit and customize to display the right information. It's nice to be able to pick and choose form features like filler text or to be able to incorporate more complex programing commands like a read only form etc. I'm not fluent with JavaScript just yet but once I am I look forward to being able to customize these forms further. I would reccomend someone at my level just using a simple html mailto: before commiting to using one of the forms since they will be easier to host.
