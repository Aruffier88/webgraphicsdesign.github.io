Illustrations with Gimp

The Following Image was Created in Gimp, See Steps Below
  1. Uploaded Crow Image to Trace
  2. Used Path Tool to Trace Crow and Create a shape for reference
  3. I liked the look of the unedited path and chose not to edit further with the design or edit tool
  4. In the gradient layer, I filled in first bird with brown with the fill bucket, used the exisiting path to fill with a gradient, then repeated in pink with the fill bucket
  5. I finally used the move tool to arrange the colored shapes
